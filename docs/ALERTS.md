## 💡 如何添加通知

### 会话如何通知你

**情况1: 紧急问题**
```markdown
在 docs/ALERTS.md 中添加：

## 🔴 [会话ID] 紧急问题标题

**时间**: 2026-03-04 HH:MM
**会话**: [会话ID]

### 问题描述
[详细描述]

### 需要老板
- [ ] 确认决策
- [ ] 提供资源
- [ ] 其他

### 处理状态
- [ ] 待处理
- [x] 已处理
```

**情况2: 需要决策**
```markdown
在 docs/ALERTS.md 的 🟡 部分添加：

## 🟡 [会话ID] 决策请求

**时间**: 2026-03-04 HH:MM
**会话**: [会话ID]

### 决策事项
[需要你决定的事情]

### 选项
- A: [选项A描述]
- B: [选项B描述]
- C: [选项C描述]

### 老板决策
- [ ] 选项 A
- [ ] 选项 B
- [ ] 选项 C
```
