## 🎨 图片设计要求

### 尺寸规格
- **推荐尺寸**: 1000 x 1500 px (2:3 比例)
- **最小尺寸**: 600 x 900 px
- **文件格式**: PNG 或 JPG
- **文件大小**: < 10MB

### 设计原则
1. **清晰标题** - 大字、易读、明确价值
2. **视觉层次** - 标题 > 产品 > 说明
3. **品牌一致** - 统一配色方案
4. **移动友好** - 大部分用户在手机浏览
5. **对比强烈** - 避免低对比度配色

### 文字可读性
- 标题: 40-60pt
- 副标题: 24-36pt
- 说明: 16-20pt
- 确保在缩略图可读

## 📝 Pin 创建流程

### 1. 图片生成
- [ ] 使用 AI 生成工具 (Midjourney/DALL-E)
- [ ] 或使用 Canva 模板
- [ ] 保存为 PNG (高质量)

### 2. Pin 设置
- [ ] 上传图片
- [ ] 添加标题（包含关键词）
- [ ] 添加描述（简洁 + CTA）
- [ ] 添加链接（Medium 文章）
- [ ] 选择板块（2-3个相关）

### 3. 发布时间
- **最佳时间**: 晚上 8-11 点 (用户活跃)
- **最佳日期**: 周末、工作日晚
- **频率**: 每天 2-3 个 Pin，分散发布
