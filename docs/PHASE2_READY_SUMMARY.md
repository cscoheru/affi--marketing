## 🔄 会话协作机制

### 通信协议

```
┌─────────────────────────────────────────────────────────────┐
│                    会话协作架构图                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌─────────────────────────────────────────────────────┐    │
│  │              docs/SESSION_REGISTRY.md (中心)          │    │
│  │              所有会话状态一目了然                      │    │
│  └─────────────────────────────────────────────────────┘    │
│                          ▲                                  │
│                          │                                  │
│           ┌──────────────┼──────────────┐                  │
│           │              │              │                  │
│  ┌────────┴────────┐ ┌───┴────┐ ┌─────┴─────┐            │
│  │  会话A (前端)   │ │会话B(后端)│ │会话C(AI) │            │
│  │  查看注册表     │ │查看注册表 │ │查看注册表│            │
│  └─────────────────┘ └──────────┘ └───────────┘            │
│           │              │              │                  │
│           └──────────────┼──────────────┘                  │
│                          │                                  │
│                          ▼                                  │
│  ┌─────────────────────────────────────────────────────┐    │
│  │         更新 SESSION_REGISTRY.md (状态变更)          │    │
│  │         或 SESSION_LOG.md (事件记录)                │    │
│  └─────────────────────────────────────────────────────┘    │
│                                                              │
│  跨会话消息: docs/MESSAGES/{目标会话}.md                    │
│  API变更: docs/API_CHANGES.md                               │
│  数据变更: docs/SCHEMA_CHANGES.md                            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 会话启动流程

```
[新会话开始]
       │
       ├─→ 1. 查看 docs/SESSION_REGISTRY.md 了解整体状态
       │
       ├─→ 2. 查看 docs/COLLABORATION_QUICK_START.md
       │
       ├─→ 3. 查看 docs/MESSAGES/{会话ID}.md 是否有留言
       │
       ├─→ 4. 查看 docs/API_CHANGES.md 和 SCHEMA_CHANGES.md
       │
       ├─→ 5. 开始工作
       │
       ├─→ 6. 完成任务后更新 docs/SESSION_LOG.md
       │
       └─→ 7. 通知依赖会话
```
