## 联盟链接创建流程

1. **登录 Amazon Associates**
   - URL: `https://affiliate-program.amazon.com`
   - 选择美国市场

2. **搜索产品**
   - 输入产品名称或 ASIN
   - 找到正确的产品（注意价格和评价数）

3. **创建链接**
   - 点击 "Text & Image"
   - 选择短链接格式
   - 复制链接

4. **记录到本文档**
   - 更新上面的表格
   - 保存短链接

## ⚠️ 注意事项

1. **不要过度链接**: 每篇文章 3-5 个链接足够
2. **链接自然融入**: 不要让链接显得太突兀
3. **明确标注**: 使用 "Amazon US" 或 "Buy on Amazon" 等明确文字
4. **测试链接**: 发布前确保所有链接都能正常打开
5. **遵守规则**: 不要使用欺骗性文字诱导点击
